sload

sload(evm)

Loads to the stack, the value corresponding to a certain key from the storage of the current account.

Parameters

evm – The current EVM frame.

Raises

  • StackUnderflowError – If len(stack) is less than 1.

  • OutOfGasError – If evm.gas_left is less than 50.

def sload(evm: Evm) -> None:
    evm.gas_left = subtract_gas(evm.gas_left, GAS_SLOAD)

    key = pop(evm.stack).to_be_bytes32()
    value = get_storage(evm.env.state, evm.message.current_target, key)

    push(evm.stack, value)

    evm.pc += 1
